** C++初识
*** 常量
    有两种定义方式
    #+begin_src C++
    #define 常量名 常量值
    const 数据类型 常量名 = 常量值
    #+end_src
    const修饰的变量也称为常量
** 数据类型
   为了给变量分配合适的内存空间
*** 整型:表示整数类型,不同区别是占用的内存空间不同
| 数据类型  | 占用空间 | 取值空间       |
| short     |        2 | $-2^{16} ~ 2^{16}$ |
| int       |        4 | $-2^{32} ~ 2^{32}$ |
| long      |        8 | $-2^{64} ~ 2^{64}$ |
| long long |        8 | $-2^{64} ~ 2^{64}$ |
*** sizeof关键字
    统计数据类型所占用的字节空间的大小

    语法: sizeof(数据类型/变量)
*** 实型
    - 单精度float
    - 双精度double
      | 数据类型 | 占用空间 | 有效数字范围 |
      | float    |        4 | 7            |
      | double   |        8 | 15~16        |
    - 默认小数是double型
    - 默认情况输出一个小数会显示6位有效数字
*** 字符型
    - 字符型变量只有一个字符,占1个字节
    - 创建字符型变量用单引号
    - 重要的ASCLL值
      - a \to     97
      - A \to    65
*** 转义字符
    | 转义字符 | 含义           | ASCLL码 |
    |----------+----------------+---------|
    | \a       | 警报           |     007 |
    | \b       | 退格           |     008 |
    | \f       | 换页           |     012 |
    | \n       | 换行           |     010 |
    | \r       | 回车           |     013 |
    | \t       | 水平制表       |     009 |
    | \v       | 垂直制表       |     011 |
    | \\       | \              |     092 |
    | \'       | '              |     039 |
    | \"       | "              |     034 |
    | \?       | ?              |     063 |
    | \0       | 0              |     000 |
    | \ddd     | 8进制转义字符  |         |
    | \xhh     | 16进制转义字符 |         |
