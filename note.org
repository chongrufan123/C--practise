** 数组
*** 概述
    - 每个元素都是相同的数据类型
    - 连续的内存组成
*** 一维数组
**** 定义：
     1. 数据类型 数组名[数组长度];
     2. 数据类型 数组名[数组长度] = {值1, 值2, ...};
     3. 数据类型 数组名[ ] = {值1, 值2, 值3 ....}
**** 数组名的作用
     1. 通过数组名统计整个数组的占用内存大小
     2. 通过数组名查看数组首地址
*** 二维数组
**** 定义
     1. 数据类型 数组名[行数][列数];
     2. 数据类型 数组名[行数][列数] = { {数据,数据} , {数据, 数据}};
     3. 数据类型 数组名[行数][列数] = {数据, 数据, 数据, 数据};
     4. 数据类型 数组名[ ][列数] = {数据, 数据, 数据, 数据};
**** 数组名的作用
     1. 通过数组名统计整个数组的占用内存大小 
     2. 通过数组名查看数组首地址

** 指针
   - 在32位操作系统指针占4个字节,64位系统指针占8个字节
   - 空指针
     #+begin_src C++
     int * p = NULL;
     #+end_src
     - 默认使用空指针对指针进行初始化
     - 内存中0-256用户没有权限访问,也就是空指针用户没办法访问
   - 野指针
     - 申请一个指针随意指向一个没有申请的地址,野指针会报错

   *空指针和野指针都不要访问*

   - const修饰指针
     1. 常量指针

        ~const int * p = &a;~
        
        指针的指向可以改,指针的值不可以改
     2. 指针常量

        ~int * const p = &a;~

        指针的指向不可以改,指针的值可以改
     3. 既修饰指针又修饰常量

        ~const int * const p = &a;~

        指针的指向和值都不可以改
   - 当 ~int * p = arr~ 时, ~p++~ 指的是指针向后偏移4个字节
